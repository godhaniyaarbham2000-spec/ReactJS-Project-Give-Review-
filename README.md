# 📝 React Comment Form Project

A clean and minimal **React + Vite Comment Form Application** with user comments, rating system, and real-time UI updates.  
Built for practice, understanding React fundamentals, and portfolio showcase.

---

## 🚀 Features

* 👤 Add username with comment
* 💬 Add remarks/comment text
* ⭐ Add rating (1 to 5)
* 📜 Display all comments instantly
* ⚡ Real-time UI updates without refresh
* 🧠 Controlled form handling in React

---

## 🛠️ Tech Stack

* React.js (Hooks: useState)
* Vite (Fast development build tool)
* JavaScript (ES6+)
* CSS3 (Custom styling)
* Local state management (React State)

## ⚙️ How to Run This Project

### 1. Clone the repository
```
git clone https://github.com/your-username/comment-form-react.git
```

### 2. Move to project folder
```
cd comment-form-react
```

### 3. Install dependencies
```
npm install
```

### 4. Start development server
```
npm run dev
```
